
**Run the Application**:
----
Clone project:
``` 
git clone https://github.com/Siencode/Flashcards-Spring-API
```
Run resource module:
```
mvnw spring-boot:run -pl resource-module
```

H2 DB console:
http://localhost:8080/h2-console

Login: sa


OpenAPi definition:
http://localhost:8080/swagger-ui.html
